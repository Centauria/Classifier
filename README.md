# Classifier
一个基于PyQt5的分类器应用，可用于有监督和无监督分类
## 概述
Classifier（下简称“本应用”）可以读取`*.csv`文件中的数据，来建立一个模型，再将这个模型用于判断输入的数据的分类。

规定`*.csv`文件的格式如下：

|属性1|属性2|……|属性n|（标签）|
|----|----|----|----|----|
|$X_{1,1}$|$X_{1,2}$|……|$X_{1,n}$|$L_1$|
|$X_{2,1}$|$X_{2,1}$|……|$X_{2,n}$|$L_2$|
|……|……|……|……|……|
|$X_{N,1}$|$X_{N,1}$|……|$X_{N,1}$|$L_N$|
